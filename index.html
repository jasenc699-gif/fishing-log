ℹ️ Data Sources
When online: Tide data is fetched from WorldTides API (requires free API key in settings). Falls back to NIWA tide tables for NZ when available.

When offline: Uses a harmonic tidal model based on an M2 semidiurnal cycle (~12h 25min period). This gives a realistic tide shape but is not calibrated to a specific port — treat as an approximation. For accurate NZ tides, visit NIWA Tides.

🎣 Moon & Fishing
ℹ️ Data Source
Moon phase is calculated mathematically using Jean Meeus's astronomical algorithms — no API required. Accuracy is within minutes and works fully offline. The calculation uses a known new moon reference date and the synodic month period (29.53059 days).

ℹ️ About Solunar Theory & Sources
Bite times are calculated from the moon's position relative to your local time using John Alden Knight's solunar theory (1926). Major periods occur when the moon is overhead or underfoot; minor periods at moonrise and moonset. The calculation is astronomical — fully accurate and works offline. Best results when solunar periods coincide with tidal movement.
